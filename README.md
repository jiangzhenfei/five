# five （线上对战五指棋）
### [线上demo](http://47.93.103.19:5900/client/)

### 线下运行
#### 1.克隆代码
#### 2.下载依赖
Using npm:
```
npm install
```
#### 3.进入server文件夹，执行
```bash
node index.js
```
#### 4.浏览器直接打开client文件夹内的index.html即可，如果想自己试一试对战，开两个浏览器窗口即可，不同用户进入同一个房间就可以尝试了
